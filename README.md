# Smart_Irrigation_System
Designed a Smart Irrigation System using ESP8266 for its Wi-Fi capability and interfaced soil moisture sensor , motor relay and water pump . Monitors soil moisture 24*7 , displays the Moisture level out of 100 , lets the user take decision based on the moisture level to switch on the pump or not . Remote monitoring and control is also possible via Blynk IoT using Wi-Fi feature in ESP8266.
